　　打字高手是一套集教学、训练、测试及网络监控于一体的五笔字型专业
培训软件，它为每一个爱好五笔的人提供一个极好的学习环境，让五笔练习
成为一种看得见、摸得着的实践活动，帮助您在极短的时间内掌握指法及五
笔输入法，是您学习的好伴侣，教学的好助手。 
　　适用于Win95/98/2000/NT操作系统及无盘Win95网。

正式发布版本功能升级情况：

1.0版功能简介
　WIN95下集教学、考核于一体的打字软件，功能完善，使用方便。
　1、功能强大的五笔字根、简码教学，指法教学；
　2、中、英文测试，实时显示速度、正确率，并自动计算综合得分；
　3、定文章，定时间，中、英文录入考核；
　4、成绩智能查询及绘制成绩曲线图；
　5、网络管理员设置；
　6、绿色软件，利于网络管理。

2.0版改进及新增功能
　1、加强五笔教学功能：增加五笔字根拆码示意图
　　　　　　　　　　　　可字根分区练习
                        显示字根编码口诀
                        增加一级简码键位图
                        换页暂停
　2、练习中可使用删除键
　3、解决重码不能录入
　4、增加背景图案
　　　并可将起动画面及背景图案更改为自己喜爱的图片
　5、自动成绩查看控制
　6、管理员口令进行加密处理
　7、自动适应６４０ｘ４８０屏幕
　8、修正1.0版中错误部分

3.5版新增功能
　1、增加背景音乐及设置功能
　2、增加背景图案快速按钮
　3、按F1键可查看多达６０多篇幅图文并茂的打字教程帮助文章
　4、动态菜单设计，可自由挂接外部程序并可设定菜单名。
　5、禁止重复运行本软件

　8、增加练习时间控制、成绩显示及保存查看
　9、五笔键名、成字字根、交叉识别码及三级简码及难字练习
　10、字体字号设置
　11、增加成绩预览及打印功能
　12、指法练习键位帮助
　13、允许最大化

4.0版新增功能
　1、wbx.dbf补充完整，可对所有汉字进行编码图示。
　2、允许修改考核评分标准。
　3、关闭时自动保存窗口状态。
　4、字根练习增设强化训练（逐步提高）项目
　5、成绩查看：可设置时间范围及排序方式。
  6、以网卡地址自动产生机位标识
　7、数据库维护：备份数据库
　　　　　　　　恢复数据库
　　　　　　　　查看数据库
　　　　　　　　合并数据库等
　8、网络监控考核过程：
　　　手动、自动刷新数据，历史记录记载，消息发送，
　　　数据维护，预览打印，远程密码修改，远程屏幕截取，
　　　桌面锁定，关闭程序及电源等。

5.0版新增功能及改正部分
　1、测试或考核中允许修改上一行打字错误。
　2、增加自由录入练习中自动换行功能。
　3、增加网络监控中锁定远程工作站桌面功能。
　4、修正多机监控时工作站只能向一台监控机发送考核时间已到的成绩记录。
　5、增加工作站向监控机发送考核成绩的校验过程。
　6、修正主控机截屏过程中不能接收成绩。
  7、在各工作站运行共享目录中wwtwin.exe可自动将当前微机NIC_ID及本机
　　编号直接添加到数据库，而不再需要管理员手工添加！
　8、增加远程测试文件传送及远程管理员设置操作。
  9、测试及练习文本颜色及文本底色可设置
 10、增加小键盘数字键录入练习
 11、为方便过滤英文半角字符，在自由录入练习中增加查找半角字符功能。
 12、管理员及监控管理员口令分开。
 13、测试前可进行输入法选择
 14、增加五笔编码查询。
 15、可任意指定其它测试文本。
 16、增加打字游戏功能。
